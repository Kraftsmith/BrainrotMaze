# Импорт недостающих моделей

Из 18 канонических брейнротов ([Brainrots.md](../../Game Design/Brainrots.md)) у нас всего 6 моделей:

| Тир | Есть | Нужно | Дыра |
|---|---|---|---|
| Common | 3 | 3 | — |
| Rare | 0 | 3 | **3** |
| Epic | 0 | 3 | **3** |
| Legendary | 1 | 3 | **2** |
| Mythic | 2 | 3 | 1 |
| Gogly | 0 | 3 | **3** |

Лабиринты Rare/Epic/Legendary/Gogly не могут спавнить контент своего тира → игрок попадает туда и видит пустоту.

Сын импортирует из Toolbox недостающие 12 моделей. Для каждой:
1. Перетащить в Workspace, проверить что есть BasePart-ы.
2. Перенести в `ServerStorage.BrainrotTemplates`.
3. Проставить атрибут `Rarity` (Common/Rare/Epic/Legendary/Mythic/Gogly).
4. (Опционально) Поставить PrimaryPart.

После — сгенерить пэды для оставшихся лабиринтов (Rare/Epic/Legendary). Скрипт уже есть, надо просто запустить с другими параметрами.
