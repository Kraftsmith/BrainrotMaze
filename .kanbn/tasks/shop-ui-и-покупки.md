# Shop UI и покупки

UI магазина для апгрейдов из [Upgrade System](upgrade-system.md). Заменяет чат-команды `/upgrade`.

## Что сделано

- `UpgradeConfig` перенесён в `ReplicatedStorage` (клиент тоже теперь читает таблицы).
- `UpgradeService` синкает уровни апгрейдов как **Player attributes** (`speedLvl`, `carryLvl`, `baseLvl`) — клиент их видит без RemoteFunction.
- `RemoteEvent ReplicatedStorage.PurchaseUpgrade` — клиент шлёт track ("speed"/"carry"/"baseCap"), сервер вызывает существующий `tryUpgrade()`.
- `StarterPlayerScripts.ShopClient` (новый LocalScript) — кнопка "🛒 Магазин [B]" в правом верхнем углу. Клик / клавиша **B** открывает панель.
- Панель: 3 строки (Скорость / Переноска / База), каждая показывает `lvl N → N+1`, текущее → следующее значение, цену в монетах. Кнопка Купить:
  - **Зелёная** — хватает монет, можно купить
  - **Красная** — не хватает (отключена)
  - **Серая "МАКС"** — достигнут максимум

Live-обновление: при изменении Coins или уровня панель сама пересчитывается.

## Как тестить (для сына)

**Что проверяем:** магазин с кнопками работает, апгрейды покупаются.

1. Нажми **Play**.
2. В правом верхнем углу появится синяя кнопка **"🛒 Магазин [B]"**.
3. Накопи минимум **100 монет** (приноси Common-брейнротов на базу, ~1.5 минуты).
4. Нажми **B** на клавиатуре (или клик по кнопке).
5. **Должна открыться панель** с тремя строками:
   - 🏃 Скорость — lvl 1→2, 16→18, 💰 100
   - 📦 Переноска — lvl 1→2, 1→2, 💰 250
   - 🏠 База — lvl 1→2, 4→6, 💰 500
6. **Цвета кнопок:**
   - Скорость зелёная (если ≥100 монет) — можно купить.
   - Переноска и База красные (если <250 / <500 монет).
7. Нажми **зелёную "Купить" у Скорости**.
8. **Должно произойти:**
   - Coins -100.
   - Кнопка ушла в строку lvl 2→3 (новая цена 300).
   - Закрой магазин (✕ или B), побеги — заметно быстрее.

### Ещё проверь

- Накопи 250, купи **Переноску** — теперь несёшь 2.
- Накопи 500, купи **База** — теперь на базу помещается 6.
- Закрытие через **✕** или нажатие **B** второй раз.
- Если открыть и не нажимать ничего, monies продолжают тикать → Coins в магазине обновляется в реалтайме.
- Достигни максимума по любому треку (купи много раз) → кнопка серая "МАКС".

### Что писать папе если что-то не так

- "Кнопка Магазин не появилась в углу" — баг ScreenGui.
- "Открыл магазин, цифр нет / только многоточия" — атрибуты не дошли с сервера.
- "Нажал Купить — Coins ушли, но эффект не применился" — UpgradeService bug.
- "Coins не уходят при покупке" — RemoteEvent не работает.
- "Магазин закрывается сам сразу после открытия" — глюк UI.
