# DataStore персистентность

`leaderstats.Coins` и состояние баз исчезают при выходе игрока. По экономике топ-апгрейды стоят 200K–1.5M монет (часы игры) — без сохранения вся прогрессия теряется и игра неиграбельна как задумано.

Использовать `DataStoreService` (или **ProfileService** для production-надёжности — автосериализация, session locking от двойного логина). Сохранять:
- Coins
- Уровни апгрейдов (скорость, переноска, база)
- Разблокированные лабиринты
- Размещённые на базе брейнроты (ID + Rarity + позиция)

Save: каждые 60 сек + on PlayerRemoving + on shutdown. Load: on PlayerAdded.

**Перед началом — закрыть [PlayerData сервис](player-data-сервис.md)** — иначе DataStore будет лезть в 5 разных мест, потом всё равно рефакторить.
