# Брейнроты

Список персонажей-брейнротов в игре, разложенный по редкостям.

Для каждого брейнрота нужно указать:
- **Имя**
- **Редкость**
- **Доход** (монет/сек, когда стоит на базе)
- **Шанс появления** (%, в лабиринте своей редкости)
- **Где спавнится** (в каких лабиринтах может встретиться)
- **Описание / внешний вид**

Принцип баланса: каждый следующий тир даёт примерно ×3–×5 дохода, но появляется в ~5–10 раз реже.

## Идеи для расширения

- Добавить **уникальные способности** для редких брейнротов (например, Bombardiro Crocodilo каждые 30 сек удваивает доход на 5 сек).
- Сделать **сезонных / ивентовых** брейнротов (Хэллоуин, Новый год) — стимул возвращаться в игру.
- Ввести **«золотые» / «радужные» версии** обычных брейнротов с шансом 1% — даёт дополнительный лут даже из ранних лабиринтов.
